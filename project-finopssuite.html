The Challenge

During my time managing $80M+ in payroll operations, I spent countless hours on tasks that should have been automated:

  • Manual data transformation: Converting messy vendor reports into standardized formats
  • Growth analysis in Excel: Building MoM/QoQ/YoY calculations from scratch
  • Executive reporting: Creating dashboards for leadership visibility
  • Data quality issues: Catching errors before they reached downstream systems

Hours of work. Every week. With room for human error at every step.

What I Learned

  • Build for real needs: Theory is nice; real operations pain points drive value
  • Data validation first: Bad data in = bad insights out. Validate early.
  • Visual feedback matters: Users trust systems that show them what's happening
  • Export is essential: Great insights trapped in a tool are useless
  • Dark mode isn't optional: Financial analysts work late. Support their eyes.
